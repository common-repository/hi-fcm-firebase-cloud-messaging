== Description ==
This plugin gives you the ability to push notifications through [Firebase Cloud Messaging](https://firebase.google.com)

**See details on GitHub:** [https://github.com/AbdallaMohammed/hi-fcm](https://github.com/AbdallaMohammed/hi-fcm)

== Installation ==
1. Copy the `hi-fcm` folder into your `wp-content/plugins` folder
2. Activate the `HI FCM - Firebase Cloud Messaging` plugin via the plugin admin page
